## ✨ ویژگی‌ها

- 📌 **پیام ثابت و تمیز**: ربات از طریق پیام‌های Inline و با ویرایش پیام اصلی عمل می‌کند تا چت همیشه مرتب و بدون شلوغی بماند.
- 📂 **دسته‌بندی‌های پویا**: دارای ۶ دسته‌بندی پیش‌فرض (فوری، کارای سالن، خریدای سالن، کارای خونه، خریدای خونه، کارای شخصی) با قابلیت اضافه و حذف کردن دسته‌بندی‌های شخصی.
- ⏰ **یادآوری زمان‌بندی شده**: تنظیم ساعت و دوره تکرار به صورت (هر روز، یک روز در میان، هفتگی).
- ✅ **دکمه‌های هوشمند**: دکمه `انجام شد` و `دیدم` برای پیام‌های یادآوری که از طریق Cron Job ارسال می‌شوند.
- 🔄 **حذف خودکار پیام‌های موقتی**: پس از انجام هر دستور، پیام‌های موقت به صورت اتوماتیک از چت پاک می‌شوند.
- 🔒 **امنیت Webhook**: پشتیبانی از قابلیت تاییدیه سکرت توکن وب‌هوک تلگرام جهت جلوگیری از درخواست‌های فیک.
- ⚡️ **معماری ماژولار**: کدهای تمیز و جدا شده برای تنظیمات، توابع و منطق اصلی.

## 🔧 فناوری‌ها و معماری

- **PHP 8+** و استفاده از Webhook برای دریافت سریع پیام‌ها.
- **SQLite**: ذخیره‌سازی سبک، سریع و بی‌نیاز به تنظیمات سرور پایگاه داده مجزا. ساختار دیتابیس در اولین اجرا به‌صورت خودکار ساخته می‌شود.
- **بدون فریم‌ورک سنگین**: کدهای خالص و بهینه شده برای اجرای سریع در سرورهای با منابع محدود.

## 🚀 نحوه اجرا و استقرار

### ۱. استقرار در سرور محلی (Local)
1. ریپازیتوری را کلون کنید:
   ```bash
   git clone https://github.com/leoson95/telegram-todo-bot.git
   cd telegram-todo-bot
   ```
2. فایل پیکربندی محیطی بسازید:
   ```bash
   cp .env.example .env
   ```
   *سپس مقادیر توکن ربات (`BOT_TOKEN`)، مسیر دیتابیس (`DB_PATH`) و توکن امنیتی وب‌هوک (`WEBHOOK_SECRET`) را تنظیم کنید.*
3. وب‌هوک ربات تلگرام خود را روی دامنه معتبری که به `index.php` اشاره دارد تنظیم کنید و اطمینان حاصل کنید که `secret_token` را هنگام تنظیم وب‌هوک ارسال می‌کنید.
4. برای فعال‌سازی یادآوری‌ها، فایل `cron.php` را در یک Cron Job با فاصله ۱ دقیقه تنظیم کنید:
   ```bash
   * * * * * php /path/to/telegram-todo-bot/cron.php
   ```

### ۲. دیپلوی روی Railway (پیشنهادی)

پروژه بهینه‌سازی شده تا به‌سادگی روی Free Tier سرویس [Railway](https://railway.app/) اجرا شود.

1. ریپو را Fork کنید.
2. در Railway یک پروژه جدید از گیت‌هاب ایجاد کنید.
3. در تب Variables متغیر `BOT_TOKEN` را با توکن ربات خود و `WEBHOOK_SECRET` را با یک رشته امن تصادفی تنظیم کنید. مسیر `DB_PATH` به صورت خودکار مقدار پیش‌فرض `/app/data/todo.db` را می‌گیرد.
4. آدرس ارائه شده توسط Railway را کپی کرده و به عنوان Webhook ربات تلگرامتان تنظیم کنید (حواستان به تنظیم پارامتر `secret_token` باشد).
5. آدرس فایل `cron.php` را در سرویس‌های رایگان مانند **[cron-job.org](https://cron-job.org)** قرار دهید تا هر ۱ دقیقه اجرا شود.

## 🔄 ساختار پروژه

```
telegram-todo-bot/
├── index.php          # نقطه ورود، دریافت درخواست‌ها (Webhook) و منطق اصلی بات
├── cron.php           # موتور یادآوری زمان‌بندی (اجرا هر ۱ دقیقه)
├── config.php         # مدیریت متغیرهای محیطی و اتصال امن به دیتابیس
├── functions.php      # توابع ارتباط با تلگرام و پردازش منطق نمایش
├── .env.example       # نمونه تنظیمات محیطی
└── README.md
```

## 🚀 نحوه استفاده

1. در تلگرام به ربات `/start` بزنید.
2. هر متنی بفرستید به عنوان کار جدید ثبت می‌شود.
3. از طریق منوی باز شده، دسته‌بندی را انتخاب کنید (می‌توانید از منوی اصلی نیز دسته‌بندی جدید بسازید).
4. اگر به یادآوری نیاز دارید، از طریق دکمه‌ها ساعت و دوره تکرار را مشخص کنید.
